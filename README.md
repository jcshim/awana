* 5억 개의 폐포가 숨을 쉬며, 
* 1,000조 개의 뇌 연결망이 작동하여 
* 나 하여금 하나님을 찬양하게 만드는 그 '통합적 설계'
* "하나님이 자기 형상 곧 하나님의 형상대로 사람을 창조하시되..." ($창세기 1:27$)
